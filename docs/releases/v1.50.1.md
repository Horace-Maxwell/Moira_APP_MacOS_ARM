# Moira 1.50.1 Release

## 版本亮点 / Release Highlights

- 修复占星盘主页面右上角相位表的线框与文字相对错位问题
- Fixes the relative misalignment between the aspect grid and its labels in the upper-right astrology panel

- 修复占星盘计算页面末尾相位表的字体与排布可读性问题
- Fixes the font and layout readability of the aspect table at the end of the astrology calculation page

- 修复多个主页面双击后右侧信息区出现空白的显示问题
- Fixes blank right-side detail panes that could appear after opening chart views from the main page

- 保留 Apple Silicon macOS 发布资产，并继续在 Release 页面提供 Windows 安装包
- Keeps the Apple Silicon macOS assets and continues to ship the Windows installer in the same release

## 下载资产 / Release Assets

- macOS 用户可以选择 `Moira-1.50.1.dmg` 或 `Moira.app.zip`
- macOS users can choose `Moira-1.50.1.dmg` or `Moira.app.zip`

- Windows 用户可以直接下载 `Moira-jre.exe`
- Windows users can download `Moira-jre.exe` directly

## 安装方式 / Install

### macOS

### 步骤 1 / Step 1

下载 `Moira-1.50.1.dmg`

Download `Moira-1.50.1.dmg`

### 步骤 2 / Step 2

打开磁盘映像

Open the disk image

### 步骤 3 / Step 3

将 `Moira.app` 拖入 `Applications`

Drag `Moira.app` into `Applications`

### 步骤 4 / Step 4

在 macOS 中正常启动应用

Launch the app normally on macOS

### Windows

### 步骤 1 / Step 1

下载 `Moira-jre.exe`

Download `Moira-jre.exe`

### 步骤 2 / Step 2

在 Windows 中运行安装包

Run the installer on Windows

- 这一版面向 Apple Silicon (`arm64`) Mac。
- This release targets Apple Silicon (`arm64`) Macs.

- 源码构建仍保持 Java 11 兼容。
- Source builds remain compatible with Java 11.

- 若未提供签名与公证凭据，本地重新打包的资产会保持可运行但不带签名公证状态。
- If signing and notarization credentials are not provided, rebuilt local assets remain runnable but are not newly signed or notarized.

- 仓库主线仍以 Apple Silicon macOS 为主，Windows 安装包作为一并提供的下载资产保留在 Release 页面里。
- The repository still focuses on Apple Silicon macOS, while the Windows installer is kept in the release assets as a convenience download.
