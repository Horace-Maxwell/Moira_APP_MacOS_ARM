# Moira 1.50.2 Release

## 版本亮点 / Release Highlights

- 修复“管理”页面默认仍跳到文字输入的问题，现在会直接显示表格视图
- Fixes the management page opening in text-entry mode, and now lands in the table view by default

- 修复“管理”页面表格网格线显示不稳定的问题，恢复白底与深灰色单元格分隔线
- Fixes inconsistent grid rendering on the management page, restoring a white table background with dark gray cell dividers

- 修复“管理”页面表格编辑时的中文输入、插入光标与提交时机问题
- Fixes Chinese input, caret visibility, and cell commit timing while editing the management table

- 修复关闭窗口时的未保存提示流程，选择保存并确认改写后会直接存档并退出
- Fixes the unsaved-changes close flow so choosing save and confirming overwrite now saves and exits immediately

- 修复 macOS 文字页面的 Command 快捷键与右键菜单提示，使全选等操作更符合系统习惯
- Fixes Command shortcuts and context-menu hints on macOS text pages, so actions like Select All follow platform conventions
