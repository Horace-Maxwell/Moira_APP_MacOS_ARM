# Security Policy

## Supported Scope

This repository focuses on the Apple Silicon macOS desktop distribution of Moira.

Security-relevant reports may include:

- code signing or notarization issues
- bundle path or writable-directory problems
- unexpected execution of external files or scripts
- release artifact integrity concerns

## Reporting

Please avoid posting secrets, certificates, passwords, or private keys in public issues.

For non-sensitive problems, open a GitHub issue with:

- affected release version
- macOS version
- Apple Silicon model
- clear reproduction steps

For sensitive reports, contact the repository owner privately through GitHub before publishing details.

## Response Goal

Reasonable effort will be made to confirm, triage, and fix valid security issues in the active Apple Silicon release line.
