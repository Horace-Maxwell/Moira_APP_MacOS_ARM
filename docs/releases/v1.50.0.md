# Moira 1.50.0 for macOS Apple Silicon

## Release Highlights

- Signed and notarized `Moira.app` and `.dmg` for Apple Silicon Macs
- Bundled arm64 Java runtime for direct end-user installation
- Safe paint-driven aspect marker overlays on macOS
- HiDPI chart rendering path for high-resolution UI mode
- Opaque, readable right-side date/time/location entry panel
- Runtime data moved to `~/Library/Application Support/Moira`

## Install

1. Download `Moira-1.50.0.dmg`
2. Open the disk image
3. Drag `Moira.app` into `Applications`
4. Launch the app normally on macOS

## Notes

- This release targets Apple Silicon (`arm64`) Macs.
- Source builds remain compatible with Java 11.
- Release assets are signed and notarized for Gatekeeper-friendly installation.
