# Moira 1.50.0 Release

## 版本亮点 / Release Highlights

- 为 Apple Silicon Mac 提供已签名、已公证的 `Moira.app` 和 `.dmg`
- Bundles signed and notarized `Moira.app` and `.dmg` releases for Apple Silicon Macs

- 内置 arm64 Java 运行时，普通用户可直接安装使用
- Includes a bundled arm64 Java runtime for direct end-user installation

- 在 macOS 上恢复了安全的、基于 paint 的角距标注线绘制
- Restores safe paint-driven aspect marker overlays on macOS

- 补齐高分屏图盘渲染链路，界面显示更清晰
- Improves the HiDPI chart rendering path for clearer high-resolution UI output

- 右侧日期、时间、地点输入面板改为不透明且更易读
- Makes the right-side date, time, and location entry panel opaque and easier to read

- 运行时可写数据迁移到 `~/Library/Application Support/Moira`
- Moves writable runtime data into `~/Library/Application Support/Moira`

- Release 页面里一并提供了 `Moira-jre.exe` Windows 安装包，方便直接查找和下载
- Publishes the `Moira-jre.exe` Windows installer alongside the macOS assets for easier discovery and download

## 下载资产 / Release Assets

- macOS 用户可以选择 `Moira-1.50.0.dmg` 或 `Moira.app.zip`
- macOS users can choose `Moira-1.50.0.dmg` or `Moira.app.zip`

- Windows 用户可以直接下载 `Moira-jre.exe`
- Windows users can download `Moira-jre.exe` directly

## 安装方式 / Install

### macOS

### 步骤 1 / Step 1

下载 `Moira-1.50.0.dmg`

Download `Moira-1.50.0.dmg`

### 步骤 2 / Step 2

打开磁盘映像

Open the disk image

### 步骤 3 / Step 3

将 `Moira.app` 拖入 `Applications`

Drag `Moira.app` into `Applications`

### 步骤 4 / Step 4

在 macOS 中正常启动应用

Launch the app normally on macOS

### Windows

### 步骤 1 / Step 1

下载 `Moira-jre.exe`

Download `Moira-jre.exe`

### 步骤 2 / Step 2

在 Windows 中运行安装包

Run the installer on Windows
